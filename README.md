# 🌐 Sany's Portfolio Website

## 📌 Overview

This is my personal portfolio website showcasing my skills, projects, and contact information. It is designed with a modern UI and interactive features to represent my journey as a CSE student and aspiring developer.

---

## 🚀 Live Demo

👉 https://mrsany.netlify.app/

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Particles.js (for animated background)

## ✨ Features

* 🔹 Responsive modern UI design
* 🔹 Animated particle background
* 🔹 Typing text effect
* 🔹 Project showcase section
* 🔹 Downloadable CV
* 🔹 Contact form layout
* 🔹 Social media integration
